### BACKEND API

This is the Backend Logic for this Ecommerce Login

## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/LoydNgei/ESHOP
    cd ESHOP
    cd API
    ```

2. Install dependencies:

    ```bash
    npm install
    ```

## Usage

To start the API, run the following command:

```bash
npm start
```
